# App Starter

Here I have included list of starter projects for different frameworks and libraries.

### WebPack

- branch: webpack-v1

### How to run the project

- Clone the repo
- Run `npm install`
- Run `npm update`
- Run `npm run dev`
- Open `http://localhost:3000/` in your browser
- Enjoy!

### How to build the project

- Run `npm run build`
